# Website Summarizer

Questo progetto utilizza il modello Ollama per analizzare il contenuto di un sito web e fornire un riassunto dettagliato in formato Markdown.

## Tecnologie Utilizzate
- Python
- BeautifulSoup
- Requests
- Ollama
